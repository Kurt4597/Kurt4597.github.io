Queue Application

Project Overview

This is a project I completed for my Programming 2 module at university. It is a Java application that uses a singly-linked list-based implementation of a Queue abstract data type (ADT) to create two separate queues of person contact information

Requirements

  • The application should possess a suitable graphical user interface (GUI)
  • the end-user can maintain two separate queues of personal contact information
  • Provide a singly-linked list-based implementation of a Queue ADT
  • Allows values to be loaded from a specified text file and inserted into the selected data structure
  • Displays appropriate messages in the message area of the GUI
